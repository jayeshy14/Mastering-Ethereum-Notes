# Mastering Ethereum Notes

This repository contains my personal notes while studying the book Mastering Ethereum.

The purpose of writing these notes is simple  
to learn the material properly and to help others who want to learn from the same book using clearer and more practical explanations.

## About the book

Mastering Ethereum is a well known book on Ethereum development written by Andreas M Antonopoulos and Gavin Wood.  
It explains how Ethereum works from fundamentals to smart contracts and decentralized applications.

## What this repository contains

* Chapter wise notes written in simple language  
* Important concepts explained with intuition  
* Code snippets and small examples  
* Personal insights while learning  

## How to use these notes

* Read alongside the original book for better understanding  
* Use the notes as quick revision material  
* Explore examples to strengthen concepts  
* Share with others who are learning Ethereum
